## 第一章：分数乘法

### 1.1 分数乘整数的意义与计算

**意义**：分数乘整数，表示求几个相同分数的和。与整数乘法的意义相同。

**计算法则**：用分子和整数相乘的积作分子，分母不变。

```
公式：a/b × n = (a × n) / b
```

**示例**：
- `2/7 × 3 = (2×3)/7 = 6/7`
- `5/12 × 4 = (5×4)/12 = 20/12 = 5/3`（约分后）

**⚠ 易错提醒**：计算结果必须约分成最简分数；能约分的先约分再计算更简便。

### 1.2 分数乘分数的意义与计算

**意义**：求一个分数的几分之几是多少。

**计算法则**：分子相乘的积作分子，分母相乘的积作分母。

```
公式：a/b × c/d = (a × c) / (b × d)
```

**示例**：
- `2/3 × 4/5 = 8/15`
- `3/4 × 2/3 = 6/12 = 1/2`

**⚠ 易错提醒**：计算中可交叉约分；分数乘分数结果通常是缩小的。

### 1.3 分数乘法的简便运算

适用整数乘法的三大运算定律：

- **乘法交换律**：`a × b = b × a`
- **乘法结合律**：`(a × b) × c = a × (b × c)`
- **乘法分配律**：`(a + b) × c = a × c + b × c`

**示例**：
- `1/2 × 3/7 × 2 = 1/2 × 2 × 3/7 = 1 × 3/7 = 3/7`（交换律）
- `2/3 × (3/4 + 1/2) = 2/3×3/4 + 2/3×1/2 = 1/2 + 1/3 = 5/6`（分配律）

## 第二章：分数除法

### 2.1 倒数的概念

**定义**：乘积为 1 的两个数互为倒数。
- 1 的倒数是 1；0 没有倒数
- 求倒数：分子分母交换位置
- 示例：`3/4` 的倒数是 `4/3`；`5` 的倒数是 `1/5`

### 2.2 分数除法的计算法则

**核心法则**：除以一个分数 = 乘这个分数的倒数。

```
a/b ÷ c/d = a/b × d/c
```

**示例**：
- `3/5 ÷ 2/7 = 3/5 × 7/2 = 21/10`
- `6 ÷ 2/3 = 6 × 3/2 = 9`

**⚠ 致命易错点**：只把**除数**变倒数，被除数不变！带分数要先化为假分数。

### 2.3 应用题类型

**类型一：已知一个数的几分之几是多少，求这个数。**
- 方法：已知量 ÷ 分率
- 例题：一桶水的 3/4 是 15 升，整桶水 = 15 ÷ 3/4 = 20 升

**类型二：求一个数是另一个数的几分之几。**
- 方法：一个数 ÷ 另一个数
- 例题：12 元是 18 元的几分之几？12 ÷ 18 = 2/3

## 第三章：比

### 3.1 比的意义

两个数相除又称这两个数的比。`a ∶ b = a/b`

比与除法、分数的对应关系：

| 比 | 除法 | 分数 |
|:--:|:----:|:----:|
| 前项 | 被除数 | 分子 |
| 比号 ∶ | 除号 ÷ | 分数线 |
| 后项 | 除数 | 分母 |
| 比值 | 商 | 分数值 |

### 3.2 基本性质与化简

**性质**：比的前项和后项同时乘或除以相同数（0 除外），比值不变。

**化简方法**：
- 整数比：`12:18 = 2:3`（同时 ÷6）
- 分数比：`2/3:4/5 = 10:12 = 5:6`
- 小数比：先扩大为整数再化简

### 3.3 按比例分配

**三步法**：求总份数 → 求每份数 → 求各部分数

**例题**：120 本图书按 3:2:1 分给三个班。
- 总份数 = 3+2+1=6；每份 = 120÷6=20
- 一班 60 本，二班 40 本，三班 20 本

## 第四章：圆

### 4.1 基本概念

- **圆心(O)**：决定圆的位置
- **半径(r)**、**直径(d)**：`d = 2r`
- 同一圆内所有半径相等，所有直径相等
- 圆是轴对称图形，有无数条对称轴

### 4.2 圆的周长

```
C = πd = 2πr   （π ≈ 3.14）
```

**示例**：半径 5cm → C = 2×3.14×5 = 31.4cm

### 4.3 圆的面积

```
S = πr²
S(环形) = π(R² - r²)
```

**示例**：半径 4cm → S = 3.14×16 = 50.24cm²

### 4.4 扇形

一条弧和经过弧两端两条半径围成的图形。面积 = n/360 × πr²。

## 第五章：百分数（一）

### 5.1 意义与互化

表示一个数是另一个数的百分之几，用 "%" 表示。**百分数不能带单位**。

| 转化方向 | 方法 | 示例 |
|:---------|:-----|:-----|
| 百分数 → 小数 | 去%，小数点左移两位 | 25% = 0.25 |
| 小数 → 百分数 | 小数点右移两位，添% | 0.36 = 36% |
| 百分数 → 分数 | 分母 100 再化简 | 75% = 3/4 |
| 分数 → 百分数 | 先化小数再化% | 1/4 = 25% |

### 5.2 生活应用

| 概念 | 公式 | 例题 |
|:-----|:-----|:-----|
| 折扣 | 现价 = 原价 × 折扣 | 200元八五折 = 170元 |
| 税率 | 应纳税额 = 收入 × 税率 | 50000×3% = 1500元 |
| 利率 | 利息 = 本金 × 利率 × 时间 | — |
