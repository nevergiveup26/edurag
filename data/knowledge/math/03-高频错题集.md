## 第一章 分数乘法

### 错题1：忘记约分导致结果错误

**原题**：计算 4/6 × 3/8

**典型错误**：`4/6 × 3/8 = 12/48 = 1/4`

**错误原因**：先乘后约分，计算量大且容易忘记约分；直接约分则简便许多。

**正确解法**：
```
4/6 × 3/8 = (4×3)/(6×8) = 12/48 = 1/4（正确但繁琐）
更好：先交叉约分：4和8约→1和2，3和6约→1和2
= (1×1)/(2×2) = 1/4
```
**同类题推荐**：6/9 × 3/4、8/15 × 5/12

---

### 错题2：乘法分配律用错

**原题**：简便计算 (1/3 + 1/4) × 12

**典型错误**：`(1/3+1/4)×12 = 1/3+1/4×12 = 1/3+3 = 3⅓`

**错误原因**：分配律不完整——只把 12 乘了 1/4，忘了乘 1/3。

**正确解法**：
```
(1/3+1/4)×12 = 1/3×12 + 1/4×12 = 4 + 3 = 7
```

**同类题推荐**：(2/5+3/10)×20、(1/2+1/3+1/6)×18

---

### 错题3：连续求几分之几时搞错单位"1"

**原题**：食堂有 120 千克大米，第一周吃了 1/3，第二周吃了剩下的 1/4。第二周吃了多少千克？

**典型错误**：`120 × 1/4 = 30`（千克）

**错误原因**：把"剩下的 1/4"当成"总数的 1/4"——单位"1"变了！

**正确解法**：
```
第一周后剩下：120 - 120×1/3 = 120 - 40 = 80（千克）
第二周：80 × 1/4 = 20（千克）
```

**同类题推荐**：一根绳子长 30 米，先用去 1/5，再用去剩下的 2/3，第二次用去几米？

## 第二章 分数除法

### 错题4：谁变倒数搞反

**原题**：计算 2/3 ÷ 4/5

**典型错误**：`2/3 ÷ 4/5 = 3/2 × 4/5 = 12/10 = 6/5`

**错误原因**：把**被除数**变成了倒数——应该变的是除数！

**正确解法**：
```
2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12 = 5/6
```
**口诀**："除号变乘号，除号后面的数变倒数"

**同类题推荐**：3/4 ÷ 5/6、7/8 ÷ 3/4

---

### 错题5：已知"具体量"和"分率"求"整体"时用错运算

**原题**：一本书看了 60 页，正好是全书的 3/5。这本书有多少页？

**典型错误**：`60 × 3/5 = 36`（页）

**错误原因**：把"已知整体求部分"和"已知部分求整体"搞混了。

**正确解法**：
```
总页数 = 看的页数 ÷ 对应的分率 = 60 ÷ 3/5 = 60 × 5/3 = 100（页）
```

**同类题推荐**：小明储蓄罐里有 45 元，占所有零花钱的 3/8，他共有多少零花钱？

---

### 错题6：带分数没有化为假分数

**原题**：计算 1½ ÷ 2/3

**典型错误**：`1½ ÷ 2/3 = 1/2 × 3/2 = 3/4`

**错误原因**：带分数不能直接取分子——必须化为假分数！

**正确解法**：
```
1½ = 3/2
3/2 ÷ 2/3 = 3/2 × 3/2 = 9/4 = 2¼
```

**同类题推荐**：2⅓ ÷ 3/5、1¼ × 2/3 ÷ 1/2

## 第三章 比

### 错题7：比值和化简比分不清

**原题**：化简比 6:9

**典型错误**：`6:9 = 2/3`

**错误原因**：2/3 是比值，不是化简比。化简比仍应写成"比"的形式。

**正确解法**：
```
6:9 = (6÷3):(9÷3) = 2:3
比值是 2÷3 = 2/3
```

**关键区分**：比值是一个数；化简比是比的形式，用":"连接。

---

### 错题8：按比例分配时总份数算错

**原题**：一块长方形地的周长是 100m，长宽比是 3:2。求长和宽。

**典型错误**：`总份数=5，长=100×3/5=60m，宽=100×2/5=40m`

**错误原因**：100m 是周长，长+宽 = 100÷2 = 50m，忘记先除以 2！

**正确解法**：
```
长+宽 = 100÷2 = 50（m）
总份数 = 3+2 = 5
长 = 50×3/5 = 30（m）
宽 = 50×2/5 = 20（m）
```

## 第四章 圆

### 错题9：周长和面积公式混淆

**原题**：半径 3cm 的圆，面积是多少？

**典型错误**：`S = 2×3.14×3 = 18.84`（cm²）

**错误原因**：用了周长公式 C = 2πr 来算面积！

**正确解法**：
```
周长：C = 2πr = 2×3.14×3 = 18.84（cm）
面积：S = πr² = 3.14×3×3 = 28.26（cm²）
```

**口诀**：周长"二π二"→ 2πr，面积"π r 方"→ πr²

---

### 错题10：环形面积中先平方还是先相减

**原题**：环形外 R=5cm，内 r=3cm，求面积。

**典型错误**：`S = 3.14×(5-3)² = 3.14×4 = 12.56`（cm²）

**错误原因**：先相减再平方——应该是**先平方再相减**！

**正确解法**：
```
S = π(R² - r²) = 3.14×(25-9) = 3.14×16 = 50.24（cm²）
```

---

### 错题11：半圆周长忘记加直径

**原题**：一个半圆的直径是 10cm，求它的周长。

**典型错误**：`C = 3.14×10÷2 = 15.7`（cm）

**错误原因**：半圆周长 = 圆周长的一半 + 直径，忘记加直径！

**正确解法**：
```
半圆周长 = πd÷2 + d = 3.14×10÷2 + 10 = 15.7 + 10 = 25.7（cm）
```

## 第五章 百分数

### 错题12：百分数不能带单位

**原题（判断）**："一根绳子用去了 30% 米"——这句话对吗？

**典型错误**：认为对

**错误原因**：百分数是比率，不能带单位！

**正确判断**：错。"30% 米"应改为"0.3 米"或"30/100 米"。

---

### 错题13：百分数化小数时小数点移错

**原题**：把 3.5% 化为小数。

**典型错误**：`3.5% = 35` 或 `3.5% = 3.5`

**错误原因**：百分号去掉后小数点移动方向和位数搞反。

**正确解法**：
```
3.5% = 3.5÷100 = 0.035（小数点左移两位）
```
**口诀**："去百分号，小数点左移两位"

---

### 错题14：折扣与"便宜了多少"混淆

**原题**：一件羽绒服原价 600 元，打六折，现价比原价便宜了多少元？

**典型错误**：`600 × 60% = 360（元）`——把现价当成了便宜的钱

**正确解法**：
```
现价 = 600×60% = 360（元）
便宜 = 600 - 360 = 240（元）
```
**或**：便宜 = 600×(1-60%) = 600×40% = 240（元）

---

### 错题15：利率问题中时间单位搞错

**原题**：存入银行 5000 元，年利率 2.5%，定期 2 年，到期利息多少？

**典型错误**：`5000×2.5% = 125（元）`——只算了 1 年的利息

**正确解法**：
```
利息 = 5000 × 2.5% × 2 = 5000×0.025×2 = 250（元）
```
**公式**：利息 = 本金 × 年利率 × 年数
