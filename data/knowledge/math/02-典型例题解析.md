## 第一章 分数乘法

### 例题1：分数乘整数

**题目**：一袋大米重 3/4 千克，5 袋这样的大米共重多少千克？

**解题步骤**：
```
3/4 × 5 = (3×5)/4 = 15/4 = 3¾（千克）
```
答：5 袋大米共重 3¾ 千克。

**解题思路**：分数乘整数，分子乘整数，分母不变，结果约分。

**易错提醒**：不要忘记约分；注意带分数表示方式。

---

### 例题2：分数乘分数

**题目**：一块长方形菜地，长 3/5 米，宽 2/3 米。这块菜地的面积是多少平方米？

**解题步骤**：
```
面积 = 长 × 宽 = 3/5 × 2/3
= (3×2) / (5×3) = 6/15 = 2/5（平方米）
```

**解题思路**：面积 = 长×宽，分子乘分子、分母乘分母，结果约分。

**易错提醒**：不可直接将分子分母分别相加！

---

### 例题3：简便运算

**题目**：用简便方法计算：7/8 × 24

**解题步骤**：
```
7/8 × 24 = 7 × 24/8 = 7 × 3 = 21
```
**简算技巧**：24 和分母 8 可以先约分（24÷8=3），再与分子 7 相乘。

**易错提醒**：整数要和分母约分，不是和分子！

---

### 例题4：连续求一个数的几分之几

**题目**：一根绳子长 20 米，先用去它的 1/4，再用去剩下的 2/5。第二次用去多少米？

**解题步骤**：
```
第一步：第一次用去 20 × 1/4 = 5（米），剩下 20-5 = 15（米）
第二步：第二次用去 15 × 2/5 = 6（米）
```
答：第二次用去 6 米。

**解题思路**：找对"单位1"——第二次的"单位1"是剩下的绳子长度，不是原来的长度。

## 第二章 分数除法

### 例题5：分数除法计算

**题目**：一个数的 2/3 是 18，这个数是多少？

**解题步骤**：
```
18 ÷ 2/3 = 18 × 3/2 = 54/2 = 27
```
答：这个数是 27。

**解题思路**：已知一个数的几分之几，求这个数 = 用已知量 ÷ 分率。

**易错提醒**：除号变乘号时，只把除数 2/3 变为 3/2！

---

### 例题6：分数除法应用题

**题目**：修一条路，已经修了 240 米，正好修了全长的 4/5。这条路全长多少米？

**解题步骤**：
```
全长 = 240 ÷ 4/5 = 240 × 5/4 = 300（米）
```
答：这条路全长 300 米。

**验证**：300 × 4/5 = 240，结果正确。

---

### 例题7：混合运算

**题目**：计算：5/6 ÷ 2/3 + 1/4 × 2

**解题步骤**：
```
5/6 ÷ 2/3 = 5/6 × 3/2 = 15/12 = 5/4
1/4 × 2 = 2/4 = 1/2
5/4 + 1/2 = 5/4 + 2/4 = 7/4 = 1¾
```

**易错提醒**：先算乘除，后算加减；不同分母要通分。

## 第三章 比

### 例题8：求比值与化简比

**题目**：求 0.8:1.2 的比值并化简。

**解题步骤**：
```
比值 = 0.8 ÷ 1.2 = 2/3
化简：0.8:1.2 = 8:12 = 2:3（同时 ÷4）
```

**注意**：比值是一个数（可以是分数），化简比仍是比的形式。

---

### 例题9：按比例分配

**题目**：用 120cm 长的铁丝做一个长方形框架，长与宽的比是 3:2。长和宽各是多少厘米？

**解题步骤**：
```
总份数 = 3+2 = 5
长+宽 = 120÷2 = 60（cm）（周长除以2得到长+宽）
每份 = 60÷5 = 12（cm）
长 = 12×3 = 36（cm）
宽 = 12×2 = 24（cm）
```
答：长 36cm，宽 24cm。

**易错提醒**：周长 ÷2 才等于长+宽，不是周长直接按比例分配！

## 第四章 圆

### 例题10：求圆的周长

**题目**：一个圆形花坛的直径是 8m，绕花坛走一圈多少米？

**解题步骤**：
```
C = πd = 3.14×8 = 25.12（m）
```

### 例题11：求圆的面积

**题目**：半径 3cm 的圆的面积是多少？

**解题步骤**：
```
S = πr² = 3.14×3×3 = 3.14×9 = 28.26（cm²）
```

---

### 例题12：环形面积

**题目**：一个环形铁片，外圆半径 10cm，内圆半径 6cm。环形面积是多少？

**解题步骤**：
```
S = π(R² - r²)
  = 3.14×(10² - 6²)
  = 3.14×(100 - 36)
  = 3.14×64
  = 200.96（cm²）
```

**易错提醒**：是先平方再相减，不是先相减再平方！

## 第五章 百分数

### 例题13：求百分率

**题目**：六（1）班有 50 人，今天 2 人请假。今天的出勤率是多少？

**解题步骤**：
```
出勤人数 = 50 - 2 = 48（人）
出勤率 = 48÷50×100% = 0.96×100% = 96%
```

---

### 例题14：折扣问题

**题目**：一双运动鞋原价 350 元，打七折出售，需要多少钱？比原价便宜多少？

**解题步骤**：
```
现价 = 350×70% = 350×0.7 = 245（元）
便宜 = 350 - 245 = 105（元）
```

---

### 例题15：税率问题

**题目**：李叔叔月工资 8000 元，扣除 5000 元免征额后按 3% 缴纳个人所得税，应缴多少税？

**解题步骤**：
```
应纳税部分 = 8000 - 5000 = 3000（元）
个人所得税 = 3000×3% = 90（元）
```

## 综合应用题

### 例题16：综合题

**题目**：一个圆形餐桌的直径是 2m。
（1）它的面积是多少？
（2）如果一个人需要 0.5m 宽的桌沿位置就餐，这张餐桌最多能坐几人？
（3）如果在餐桌周围包一圈铝合金边条，需要多长的边条？

**解题步骤**：
```
（1）S = πr² = 3.14×1×1 = 3.14（m²）
（2）C = πd = 3.14×2 = 6.28（m）
    人数 = 6.28÷0.5 ≈ 12.56，取整 ≈ 12（人）
（3）边条长 = C = 6.28（m）
```
