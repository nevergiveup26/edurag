# EduRAG 智慧问答系统 — 操作指南

> 适用版本：v1.0 | 适用角色：学生、管理员

---

## 一、系统概述

EduRAG 是一套面向 K12 教育的智能问答系统，基于 RAG（检索增强生成）技术，结合知识库管理、多模态识别、作业批改、错题管理等功能，为学生提供个性化的 AI 学习助手体验，为管理员提供知识库维护和质量评估工具。

系统分为两大端：

- **学生端**：智能问答、作业批改、拍照搜题、错题集（艾宾浩斯复习）、对话历史
- **管理端**：仪表盘、知识库管理、文档上传、检索可视化、调用链追踪、RAG 评测

## 二、学生端操作指南

### 2.1 登录

访问系统首页后，进入登录页面，选择 **学生** 角色，输入用户名和密码登录。如还没有账号，点击"注册"链接创建新账号。

登录成功后自动跳转到学生端主页（智能问答页面）。

### 2.2 智能问答

**功能说明**：向 AI 学习助手提问，系统会自动检索知识库中的相关内容，结合大语言模型生成准确回答。

**操作步骤**：

1. 在页面顶部确认当前选中的 **学科**（数学 / 语文 / 英语），系统会根据所选学科限定检索范围
2. 在底部的输入框中输入你的问题，例如：
   - 「圆的面积公式是什么？」
   - 「请解释《草原》这篇课文的中心思想」
   - 「What is the past tense of 'go'?」
3. 按 **回车键** 或点击右侧的 **发送按钮**（蓝色箭头图标）
4. 系统会展示思考过程（思考链）——包括调用哪些工具、检索了哪些内容等
5. 等待 AI 生成回答，回答采用 Markdown 格式渲染（支持代码块、表格、公式等）

**提示**：
- 问题越具体，回答越准确
- 可以在对话中连续追问，系统支持多轮对话
- 如果想换一个话题，可以点击侧边栏"历史记录"新建对话

### 2.3 作业批改

**功能说明**：提交题目内容和你的作答，系统会自动批改并给出详细反馈（分步评分、错误标注、改进建议）。答错的题目会自动收录到错题集。

**操作步骤**：

1. 在页面顶部切换到「📝 作业批改」标签页
2. **第一步 — 填写题目信息**：选择学科（数学/语文/英语）、题型（客观题/主观题）、年级（小学/初中/高中）
3. **第二步 — 输入题目**：在文本框中输入题目内容，也可以点击右侧的「📷 拍照」按钮上传题目图片
4. **第三步 — 输入作答**：在文本框中输入你的答案，也可以拍照上传作答图片
5. 确认信息无误后，点击底部蓝色的「✅ 提交批改」按钮
6. 等待系统完成批改，查看结果：
   - **总分**：右上角显示得分（绿色≥60分/红色<60分）
   - **反馈**：整体评价和建议
   - **分步评分**：每步的得分和进度条，正确✓ / 错误✗ / 部分正确△
   - **错误标注**：指出具体错误位置、正确解法、错误原因
   - **改进建议**：学习建议

**自动收录**：得分低于 60 分或不正确的题目会自动保存到错题集，方便后续复习。

### 2.4 拍照搜题

**功能说明**：拍摄或上传题目图片，系统通过多模态识别和 RAG 检索匹配相关题目和解答。

**操作步骤**：

1. 点击侧边栏「📷 拍照搜题」进入页面
2. 上传题目图片：点击上传区域选择图片，或直接拖拽图片到上传区（支持 JPG/PNG，不超过 10MB）
3. （可选）在文本框中输入补充描述
4. 点击「🔍 开始搜题」或「📝 文字搜题」
5. 查看搜索结果——每条结果包含：
   - 相关度百分比
   - 匹配题目
   - 参考解答

### 2.5 错题集

**功能说明**：查看和管理所有答错的题目，利用艾宾浩斯遗忘曲线科学安排复习计划。

**主要功能模块**：

| 模块 | 说明 |
|------|------|
| 遗忘曲线图 | 上方展示艾宾浩斯理论曲线，下方散点为你的错题分布，红色标记待复习题目 |
| 复习时间表 | 显示 6 次复习节点：1 天 / 2 天 / 4 天 / 7 天 / 15 天 / 30 天 |
| 统计卡片 | 总错题数、待复习数、涉及学科数 |
| 学科过滤 | 顶部下拉选择学科，只显示该学科的错题 |

**操作步骤 — 查看错题详情**：
每条错题展示题目、你的作答 vs 正确答案、批改结果（分步评分、错误标注）。

**操作步骤 — 举一反三**：
1. 在错题列表中，点击某条错题的「💡 举一反三」按钮
2. 等待 AI 生成 3 道变式题
3. 展开每道变式题，可查看提示（可折叠）和参考答案（可折叠）

**操作步骤 — 标记已复习**：
1. 点击「✔ 标记已复习」按钮
2. 系统更新该错题的复习计数，下次复习时间按艾宾浩斯曲线顺延

### 2.6 对话历史

**功能说明**：查看和管理所有历史对话记录。

**操作步骤**：

1. 点击侧边栏「🕐 历史记录」进入页面
2. 查看对话列表（标题、消息数、创建时间）
3. 点击某条对话查看详细消息内容（右侧抽屉滑出）
4. 点击「🗑 删除」可删除不需要的对话
5. 点击「新建对话」跳转到智能问答页面

## 三、管理端操作指南

### 3.1 登录

在登录页面选择 **管理员** 角色，输入管理员用户名和密码登录。管理员账号需要由系统管理员预先创建。

### 3.2 仪表盘

**路径**：侧边栏 → 📊 仪表盘

**功能说明**：查看系统整体运行状况。

**操作步骤**：

1. 登录后自动进入仪表盘
2. 查看 4 张统计卡片：文档总数、知识块数、用户数、总查询量
3. 查看查询趋势图表（近 7 天折线图）和文档类型分布图
4. 查看最近上传文档表格

### 3.3 知识库管理

**路径**：侧边栏 → 📚 知识库

**功能说明**：创建和管理知识库，将文档归入不同知识库。知识库按学科（数学/语文/英语）分类，学生提问时会根据所选学科限定检索范围。

**操作步骤 — 创建知识库**：

1. 点击左侧面板顶部的「+ 新建知识库」按钮
2. 填写知识库信息：
   - **名称**（必填）：如「六年级数学上册」
   - **描述**：如「人教版六年级上册数学知识点汇总」
   - **分类**（选填）：数学 / 语文 / 英语 等
   - **标签**（选填）：如 六年级、上册、人教版
3. 点击确认创建

**操作步骤 — 向知识库添加文档**：

1. 在左侧面板点击选中目标知识库
2. 点击「添加文档」按钮
3. 在弹出的文档选择器中勾选要加入的文档
4. 点击确认（也可以在上传文档时直接选择目标知识库，见下一节）

**操作步骤 — 从知识库移出文档**：

1. 选中知识库，在右侧的文档表格中点击「移除」

**操作步骤 — 删除知识库**：

1. 选中知识库，点击右侧底部的「删除知识库」按钮（红色）

> ⚠️ 系统启动时会自动创建「数学」「语文」「英语」三个默认知识库，无需手动创建。

### 3.4 文档上传

**路径**：侧边栏 → 📤 文档上传

**功能说明**：上传教学文档到系统。系统会自动完成文档加载 → 文本切分 → 向量化 → 存储，使文档内容可被检索。

**支持格式**：PDF、DOCX、TXT、Markdown、HTML、图片（JPG/PNG/GIF/WebP/BMP）

**操作步骤**：

1. （可选）在页面顶部的「📁 目标知识库」下拉框中选择一个知识库，上传后文档会自动关联到该库
2. 点击或拖拽文件到上传区域（支持批量上传）
3. 确认文件列表无误后，点击「📤 上传 N 个文件」按钮
4. 等待上传完成，每个文件会显示成功 ✓ 或失败 ✗ 状态
5. 上传成功的文档会出现在下方「已上传文档」表格中

**操作步骤 — 预览文档内容**：

1. 在已上传文档表格中，点击某条文档的「👁 预览」按钮
2. 弹窗展示文档的完整文本内容
3. 点击「关闭」退出

**操作步骤 — 删除文档**：

1. 在已上传文档表格中，点击某条文档的「🗑 删除」按钮
2. 弹出确认框，点击「确定」
3. 文档从 MySQL、向量库和本地存储中一并移除

> ⚠️ 删除操作不可逆，请确认后再操作。

### 3.5 检索可视化

**路径**：侧边栏 → 🔍 检索可视化

**功能说明**：输入查询文本，实时查看系统的检索结果，用于验证知识库内容是否被正确检索。

**操作步骤**：

1. 查看顶部的统计卡片了解当前索引规模
2. 在输入框中输入测试查询，例如「什么是分数乘法」
3. 系统展示 Top-K 条检索结果，每条包含相关度百分比、来源文档和内容片段

### 3.6 调用链追踪

**路径**：侧边栏 → 🔗 调用链追踪

**功能说明**：查看每次 RAG 查询的完整执行链路（基于 Langfuse），用于定位问题或优化策略。

**操作步骤**：

1. 在列表中查看最近查询记录：Trace ID、查询内容、策略、耗时、状态
2. 耗时颜色分级：绿色（<1s）/ 黄色（1-3s）/ 红色（>3s）
3. 点击某条记录查看详情弹窗：
   - 基本信息：查询文本、策略、总耗时
   - 执行步骤时间线
   - 检索结果（每条的来源文档和相关度）

### 3.7 RAG 检索评估

**路径**：侧边栏 → 📈 RAG 评估

**功能说明**：运行标准检索评估，计算 Precision、Recall、F1、MRR、NDCG、Hit Rate 等指标，用于衡量系统检索质量。

**操作步骤**：

1. 点击「▶ 运行评测」按钮
2. 实时观察进度条（当前/总样本数 + 耗时计时器）
3. 查看实时日志面板（SSE 事件流）
4. 评测完成后查看指标卡片：Precision、Recall、F1 Score、MRR、NDCG、Hit Rate、关键词匹配率、平均执行时间
5. 查看 ECharts 图表和样本详情表格
6. 可点击「⏹ 取消」随时终止评测

### 3.8 RAGAS 评测

**路径**：侧边栏 → 📊 RAGAS 评测

**功能说明**：运行 RAGAS 标准评测框架，评估答案质量（忠实度、相关性、正确性等）。

**操作步骤**：

1. 点击「▶ 运行 RAGAS 评测」按钮
2. 观察两阶段进度：样本采集 → 计算评分
3. 评测完成后查看 7 项指标：Faithfulness（忠实度）、Answer Relevancy（答案相关性）、Context Relevancy（上下文相关性）、Context Precision（上下文精确率）、Context Recall（上下文召回率）、Answer Correctness（答案正确性）、Avg Score（平均分）
4. 查看评测详情表格（每个样本各项得分）

### 3.9 评估历史

**路径**：侧边栏 → 🕐 评估历史

**功能说明**：查看、对比和导出历史评测记录。

**操作步骤 — 查看详情**：

1. 在列表中点击某条记录，右侧展示评测详情
2. 可查看指标数值、图表和样本报告

**操作步骤 — 对比两次评测**：

1. 勾选 2 条记录（多选复选框）
2. 点击「📊 对比」按钮
3. 查看各项指标的差值（正数=进步，负数=退步）

**操作步骤 — 导出 PDF 报告**：

1. 在详情页面点击「📄 导出 PDF」
2. 浏览器自动下载评估报告 PDF（含中文字体、指标表格和图表）

**操作步骤 — 删除历史记录**：

1. 点击某条记录的「🗑 删除」按钮

## 四、常见问题 (FAQ)

### Q1：学生提问时为什么回答不准确？
- 确认已选择正确的学科（数学/语文/英语）
- 确认对应知识库中有相关文档（管理员需要先上传该学科的知识文档）
- 尝试将问题表述得更具体

### Q2：上传的文档为什么搜不到？
- 文档上传后需要确保已关联到正确的知识库
- 进入「检索可视化」页面输入文档中的关键词测试检索效果
- 检查文档格式是否支持（PDF/DOCX/TXT/MD/HTML）

### Q3：删除的文档为什么会"复活"？
- 确保删除操作返回"删除成功"提示
- 如果删除时报错但文档已消失，说明 MySQL 已删除成功但向量库或本地存储可能残留，重启服务后会生效
- v1.0.1 已修复此问题，删除时同步清理所有存储层

### Q4：作业批改分数准确吗？
- 系统使用 AI 模型批改，建议作为参考，重要考试请以老师评分为准
- 批改结果会展示详细的分步评分和错误标注，便于对照学习

### Q5：如何获取更好的评测结果？
- 定期上传高质量的教学文档到对应知识库
- 运行 RAG 评估查看检索质量指标
- 运行 RAGAS 评测查看答案质量
- 通过调用链追踪定位慢查询或不相关检索

### Q6：如何备份数据？
- 文档内容存储在 MySQL 数据库中，定期备份 MySQL 即可
- 向量数据存储在 Milvus 中，建议同时备份 Milvus 数据
- 本地 chunk 存储在 `data/chunks_index.json`，可一并备份
